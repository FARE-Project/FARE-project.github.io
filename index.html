Firmware Analysis for vulneRability dEtection

The project aims at devising novel static and dynamic analysis techniques able to identify portions of code that may contain software vulnerabilities in embedded systems.

Project FARE (202225BZJC, D53D23008380006) is funded by Piano Nazionale di Ripresa e Resilienza (PNRR) – Missione 4 “Istruzione e Ricerca” – Componente 2 "Dalla Ricerca all'Impresa' Investimento 1.1 "Fondo per il Programma Nazionale di Ricerca e Progetti di Rilevante Interesse Nazionale (PRIN)" (Avviso D.D 104 02.02.2022) - European Union - NextGenerationEU.
